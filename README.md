# Asia Pacific Local Governance Project Mapping Dashboard

## Overview

This project demonstrates the design and development of an interactive Power BI dashboard for mapping local governance and development projects across Asia Pacific using publicly available UNDP project data.

The dashboard supports portfolio oversight, donor visibility, SDG alignment tracking, geographic analysis, and evidence-based decision making through interactive visualizations, filtering, and drill-through reporting.

---

## Project Objectives

The dashboard was developed to:

- Visualize local governance and development projects across Asia Pacific.
- Track project budgets, donors, thematic focus areas, and SDG alignment.
- Enable geographic exploration of project locations.
- Support portfolio-level and project-level analysis.
- Demonstrate modern Power BI dashboard design and reporting practices.
